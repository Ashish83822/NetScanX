# NetScanX

## Overview
NetScanX is a Python-based network scanning tool designed to identify open TCP ports on a target host. It is built for educational purposes and to help students learn the fundamentals of network reconnaissance.

## Features
- Scan a target IP address
- Detect open TCP ports
- Display scan results
- Save scan reports
- Beginner-friendly codebase

## Technologies Used
- Python 3
- Git
- Visual Studio Code
